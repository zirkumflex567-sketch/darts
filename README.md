# darts

Dartsmind MVP als React-Native-(Expo)-App mit Web-Unterstuetzung.

## Schnellstart (VPS)
```bash
npm install
npm run dev:web
```

Weitere Kommandos:
```bash
npm run dev
npm run dev:tunnel
npm run dev:client
npm test
npm run lint
```
